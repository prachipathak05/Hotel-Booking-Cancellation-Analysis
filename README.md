# Hotel-Booking-Cancellation-Analysis

## 📌 Project Overview
This project focuses on analyzing hotel booking data to understand the factors that influence booking cancellations. The analysis is performed using Python and various data visualization techniques to extract meaningful insights.

---

## 🎯 Objectives
- Identify patterns in booking cancellations  
- Analyze customer behavior and booking trends  
- Determine key factors affecting cancellations  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas    
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## 📊 Dataset
The dataset used in this project is sourced from Kaggle and contains detailed information about hotel bookings, including customer details, booking status, and other relevant features.

---

## 🔍 Key Analysis
- Data cleaning and preprocessing  
- Handling missing values  
- Exploratory Data Analysis (EDA)  
- Visualization of cancellation trends  
- Analysis based on lead time, customer type, and country  

---

## 📈 Key Insights
- Bookings with longer lead times are more likely to be cancelled  
- Certain customer types show higher cancellation rates  
- Portugal has the highest number of booking cancellations  

---

## 📁 Project Files
- `Hotel_Booking_Analysis.ipynb` – Jupyter Notebook  
- `Hotel_Booking_Report.pdf` – Project Report  

---

## ✅ Conclusion
This analysis provides valuable insights into customer booking behavior and highlights factors that contribute to cancellations. These insights can help hotels make data-driven decisions to reduce cancellations and improve operational efficiency.
